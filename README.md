# Fengard — rete delle compresenze

Rete interattiva costruita a partire dall'archivio ForumCommunity e da trascrizioni locali revisionate.

Aprire `index.html` oppure pubblicare la root di questa cartella con GitHub Pages.

La cartella `data/` contiene dati aggregati della rete. `sources/` contiene esclusivamente le trascrizioni locali collegate alle nuove giocate pubblicate; documenti originali, cache, lettere, compilazioni escluse e report interni non sono inclusi.
